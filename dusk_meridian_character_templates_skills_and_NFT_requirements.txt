## **NFT Tier System Summary**

### **Tier 1: Universal Role License NFTs (Required - Choose One)**
- **Guardian License** - Tank/Protector archetype (Soul-bound after first use)
- **Striker License** - Damage dealer archetype (Soul-bound after first use)
- **Specialist License** - Utility/Technical archetype (Soul-bound after first use)
- **Coordinator License** - Support/Leadership archetype (Soul-bound after first use)

### **Tier 2: Power Source Certification NFTs (Required - Choose One or More)**
- **Martial Artist Certification** - Physical combat and weapons (Tradeable)
- **Spellcaster Certification** - Elemental magic and arcane arts (Tradeable)
- **Technician Certification** - Technology, gadgets, and engineering (Tradeable)
- **Divine Channeler Certification** - Faith-based healing and support (Tradeable)
- **Nature Walker Certification** - Natural/primal forces and animals (Tradeable)
- **Psion Certification** - Mental powers and meditation (Tradeable)

### **Tier 3: Specialization NFTs (Optional - Advanced Options)**
- Specific combinations like Pyromancer, Battlemage, Shadowblade, Combat Medic
- Require specific Tier 1 + Tier 2 combinations
- Fully tradeable

### **Tier 4: Elite/Specialty NFTs (Future - Not Yet Available)**
- Advanced rare classes like Necromancer
- Not included in basic templates

## **Backend Validation Logic**

```csharp
public class CharacterTemplateValidator
{
    public async Task<ValidationResult> ValidateTemplateAccess(
        string walletAddress, 
        string templateId)
    {
        var template = CharacterTemplates.GetById(templateId);
        var ownedNFTs = await BlockchainService.GetOwnedNFTs(walletAddress);
        
        // Check Tier 1 requirement
        if (!ownedNFTs.Any(nft => nft.Type == template.RequiredTier1Role))
        {
            return ValidationResult.Failure(
                $"Missing required {template.RequiredTier1Role} License NFT"
            );
        }
        
        // Check Tier 2 requirements
        foreach (var requiredPowerSource in template.RequiredTier2PowerSources)
        {
            if (!ownedNFTs.Any(nft => nft.Type == requiredPowerSource))
            {
                return ValidationResult.Failure(
                    $"Missing required {requiredPowerSource} Certification NFT"
                );
            }
        }
        
        // Check optional Tier 3 specializations
        var availableSpecializations = new List<string>();
        foreach (var spec in template.OptionalTier3Specializations)
        {
            if (ownedNFTs.Any(nft => nft.Type == spec))
            {
                availableSpecializations.Add(spec);
            }
        }
        
        return ValidationResult.Success(
            availableSpecializations,
            ownedNFTs.Count(nft => nft.Tier >= 1 && nft.Tier <= 3)
        );
    }
}
```

This system ensures that character templates are properly gated by NFT ownership while providing clear progression paths for players to unlock more advanced combinations.
